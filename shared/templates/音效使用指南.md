# 🎵 音效使用指南

## 安全原则

**API Key 永不写入代码或配置文件。**

ElevenLabs API Key 通过环境变量传入：

```bash
# 正确用法
ELEVENLABS_API_KEY=sk-xxx bash scripts/generate_audio.sh projects/饼干碎了_ItsOkay

# 或在 shell 中提前 export（不会打印到日志）
export ELEVENLABS_API_KEY
bash scripts/generate_audio.sh projects/饼干碎了_ItsOkay
```

**脚本中通过 `${ELEVENLABS_API_KEY:-}` 读取，`:-` 表示变量不存在时返回空字符串，不会报错也不会泄露。**

## 双方案支持

| 方案 | 条件 | 说明 |
|------|------|------|
| ElevenLabs | 设置了 `ELEVENLABS_API_KEY` | 自动生成 BGM 和音效 |
| Freesound | 未设置 key | 提供搜索链接，手动下载 |
